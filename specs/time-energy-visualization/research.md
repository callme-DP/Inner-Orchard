# 时间与精力可视化调研（Feat-001）

## 已知
- ActivityWatch：可通过本地 API 拉取 bucket 事件；需确认 aw-watcher-web 安装与权限。
- 日历：后续从 macOS Calendar 读取（JXA 或 EventKit），按日历名称映射类别（含“突发任务”）。
- Mood：可由 MoodExample 导出或 `scripts/log_mood.py` 生成本周/上周记录。

## 待确认
- 日历类别映射表：工作/沟通/阅读/健康/娱乐/突发任务（需要最终对照表）。
- 突发任务口径：是否仅统计“突发任务”日历下的事件数，还是需要附带持续时长。
- 粒度落地：月/年视图的对比口径（对比上一月/上一年）及占位策略。

## 后续验证
- 渲染性能：单页加载大数据集时的表现（可按需做惰性加载）。
- 校验规则：耗时=事件合计，缺段提示；跨时区/无权限时的降级文案。
